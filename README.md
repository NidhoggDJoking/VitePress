## [VitePress ](https://vitepress.docschina.org/)


### [`Gitee`仓库镜像功能](https://gitee.com/help/articles/4336)

目前已使用`Gitee`所支持设置的两种镜像：

- Push：用于将 `Gitee` 的仓库自动镜像到 `GitHub` 。
- Pull：用于将 `GitHub` 的仓库镜像到 `Gitee` 。
